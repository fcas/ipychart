### Install

``` bash
# using pip
$ pip install ipychart
```

::: warning COMPATIBILITY NOTE
As it is a [Jupyter Widget](https://ipywidgets.readthedocs.io/en/stable/), ipychart only works in a [Jupyter](https://jupyter.org/) environment
:::
